# freeipa_sudo_rule_allowcmd_membership (Resource)



## Example Usage

```terraform
resource "freeipa_sudo_rule_allowcmd_membership" "allowed_cmd" {
  name    = "sudo-rule-editors"
  sudocmd = "/bin/bash"
}

resource "freeipa_sudo_rule_allowcmd_membership" "allowed_cmd" {
  name       = "sudo-rule-editors"
  sudocmds   = ["/bin/bash"]
  identifier = "allowed_bash"
}
```
